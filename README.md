# Vim Handbook

《Vim Handbook》是一本旨在帮助 Vim 初学者学习了解使用 Vim 的入门书。

本书将分为四个部分：

1. 第一部分适用于那些想要了解并简单使用Vim的新用户如何在Vim中生存下去，列如如何退出Vim
1. 第二部分面向对Vim已经有一定了解的朋友更好的使用Vim的功能及使用插件来提升效率
1. 第三部分将学习Vim Script
1. 第四部分将从头编写一个完整Vim插件
